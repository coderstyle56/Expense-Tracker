Expense Tracker

A simple Expense Tracker application built using HTML, CSS, and JavaScript. This application helps users keep track of their expenses without needing a database, as it uses the browser's local storage for data persistence.

Features-
Add Expenses: Input expense details like description, amount, and date.
Edit Expenses: Update existing expense entries.
Delete Expenses: Remove expense entries.
Total Expense Calculation: Automatically calculates and displays the total expenses.
Local Storage: Saves and retrieves expense data using local storage.

Technologies Used:
1.HTML
2.CSS
3.JavaScript
